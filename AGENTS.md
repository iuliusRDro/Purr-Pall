- **Antigravity (Local):** Lead Developer & Git Administrator. Handles "Deep Reasoning", security, and local architecture.
- **Jules (Remote):** Integration Partner. Handles PRs, code review, and remote synchronization.

## 4. WORKFLOW PROTOCOLS

- **Branching:**

  - Fix branches: `fix/issue`

- **Commits:** Frequent local commits.
- **Handoff:** Update `HANDOFF.md` before passing to the next agent.
- **Merge Strategy:** "Squash and Merge" only.
